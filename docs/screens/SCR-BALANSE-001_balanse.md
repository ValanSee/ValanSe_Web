## 🚪 진입 경로

- 하단 네비게이션 "밸런스" 탭

### 3. BalanceList

**표시 데이터**: 페이지네이션된 게임 목록

**사용 API**: `fetchVotes`

**사용자 액션**

- 카드 클릭 → `/poll/[id]`
- 스크롤 → 다음 페이지 로드
- 관리자: PinButton

**상태별 처리**

- **로딩**: 스켈레톤 카드
- **빈 상태**: "해당 카테고리에 게임이 없어요"
- **에러**: 재시도 버튼
