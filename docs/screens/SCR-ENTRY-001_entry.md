- 인가 후 `/oauth/kakao/redirect`로 콜백

## 🔄 사용자 시나리오

**시나리오 1: 최초 로그인**

1. `/entry` 진입
2. 카카오 로그인 버튼 클릭
3. 카카오 인가 → 콜백 → 신규면 `/onboarding`, 기존이면 `/main`
