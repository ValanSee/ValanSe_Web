## 🧩 섹션별 상세 명세

### 0. 프로필 이미지

**사용자 액션**

- 아바타(카메라 뱃지) 탭 → 파일 선택 → 선택 즉시 `PUT /member/profile-image` 업로드
- 저장 버튼과 무관하게 독립적으로 동작 (서버가 기존 이미지 삭제 후 교체)

**상태별 처리**

- **검증 실패 (업로드 전 클라이언트 검증)**: jpeg/png/webp/gif 외 형식 또는 5MB 초과 시 alert, API 호출 없음
- **업로드 중**: 아바타 위 반투명 오버레이 + 스피너, 재선택 비활성화
- **성공**: 응답 `profile_image_url`로 store(`mypageData`) 갱신 → 아바타 즉시 교체
- **실패**: alert 후 기존 이미지 유지
