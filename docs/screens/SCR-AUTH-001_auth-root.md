## 🔄 사용자 시나리오

**시나리오 1: 정상 로그인 사용자**

1. 사용자가 `/` 진입
2. `fetchProfileThunk`로 프로필 조회 성공
3. `/main`으로 자동 이동

**시나리오 2: 토큰 만료 / 미로그인**

1. 사용자가 `/` 진입
2. 프로필 조회 실패 (401 등)
3. `/entry`로 자동 이동

**시나리오 3: 신규 가입자**

1. 로그인 직후 `/` 진입
2. 프로필이 있으나 온보딩 미완료
3. `/onboarding`으로 자동 이동
