## 🎯 화면 목적

특정 밸런스 게임의 선택지·통계·댓글을 보고, 투표 및 댓글 작성에 참여한다. 관리자는 핀 고정·삭제 기능을 사용한다.

## 🧩 섹션별 상세 명세

### 1. PollCard

| 항목 | 출처 | 비고 |
|---|---|---|
| 제목 | `GET /votes/{id}` | - |

**사용자 액션**

## 🔄 사용자 시나리오

**시나리오 1: 미참여 사용자 투표**

1. `/poll/[id]` 진입
2. 선택지 A 클릭
3. 결과 차트 노출 + 댓글 영역 활성화

**시나리오 2: 관리자 핀 고정**

1. 관리자가 진입
2. FAB → "핀 고정" 선택
3. `pinVote` 호출 → 토스트

## ⚠️ 예외 상황

| 상황 | 처리 |
|---|---|
| 존재하지 않는 id | not found 메시지 + 이전 화면으로 |
| 이미 투표한 게임 | 선택지 비활성화, 결과 즉시 노출 |
| 미로그인 + 투표 시도 | `/entry`로 유도 |

## 🔌 사용 API

| Method | Endpoint | 용도 |
|---|---|---|
| GET | `/votes/{id}` | 상세 조회 |
| GET | `/votes/{id}/comments` | 댓글 목록 |
| GET | `fetchBestComment` | 베스트 댓글 |
| GET | `fetchBestVote` | 통계 |
| POST | `/votes/{id}/comments` | 댓글 작성 |
| PUT | `/votes/{id}/pin` | 핀 고정 (관리자) |
| DELETE | `/votes/{id}` | 게시글 삭제 (관리자) |
